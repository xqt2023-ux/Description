## 使用方法

在对话中直接提及 skill 的用途，Claude 会自动识别并使用相应的 skill。例如：

```
"帮我创建一个技术文档" → 使用 docx skill
"设计一个登录页面" → 使用 frontend-design skill
"分析这个 Excel 数据" → 使用 xlsx skill
"创建一个思维导图" → 使用 json-canvas skill
```

## 查看 Skill 详情

每个 skill 的详细说明在其对应目录的 `SKILL.md` 文件中：

```bash
cat .claude/skills/<skill-name>/SKILL.md
```

## 更新 Skills

要更新所有 skills 到最新版本：

```bash
# 克隆最新版本
git clone --depth 1 https://github.com/anthropics/skills.git /tmp/anthropic-skills

# 复制到项目
cp -r /tmp/anthropic-skills/skills/* .claude/skills/
```
